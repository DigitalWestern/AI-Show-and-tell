# 04 Scoring Matrix

Use this file after cross-examination. Scores should reflect evidence quality, not how interesting the idea feels.

## Scoring Rules

- Score each category from 1 to 5.
- Weight each category so total weight equals 100%.
- Multiply score by weight to get weighted score.
- A missing evidence category should usually score 1 or 2.
- The final recommendation cannot exceed the evidence-quality ceiling.

## Evidence-Quality Ceiling

Use the weakest important evidence area to cap the recommendation.

| Weak area | Why it matters | Ceiling |
| --- | --- | --- |
| No current prices / yields / spreads | Cannot prove relative value in public credit | Proceed to diligence |
| No covenant detail | Cannot assess lender protection | Proceed to diligence |
| No downside scenario | Cannot underwrite loss case | Proceed to diligence |
| No source-backed demand bridge | Cannot prove revenue recovery | Watchlist |

## Recommendation Bands

These are guides, not automatic outputs.

| Weighted score | Usual recommendation |
| ---: | --- |
| 1.0-1.9 | Reject |
| 2.0-2.9 | Watchlist |
| 3.0-3.9 | Proceed to diligence |
| 4.0-5.0 | Invest / underwrite, only if evidence ceiling allows it |
