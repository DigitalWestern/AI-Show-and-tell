# 03 Cross-Examination

Use this file after the independent outputs are complete. The goal is to force disagreement into the open before the final memo.

## Rules

- No new outside facts unless they are added to the evidence pack first.
- Critiques must cite the claim being challenged.
- Generic objections do not count.
- Each agent must identify what evidence would change their mind.

## Cross-Examination Prompts

### Bear Analyst Attacks Bull Case

Prompt:

Review the Bull Analyst output and identify the three most fragile assumptions. For each one, explain why the evidence may not support the claim, what downside scenario follows if the assumption is wrong, and what diligence would prove or disprove it.

Output:

| Bull claim challenged | Bear critique | Evidence gap | Diligence needed |
| --- | --- | --- | --- |
|  |  |  |  |

### Bull Analyst Responds

Prompt:

Respond to the Bear Analyst critique. Concede weak points where appropriate. Defend only claims supported by the evidence pack.

Output:

| Bear critique | Bull response | Conceded? | What would settle it |
| --- | --- | --- | --- |
|  |  | Yes / No / Partial |  |

### Credit Analyst Separates Equity From Credit

Prompt:

Review the Bull and Bear cases. Explain whether the opportunity is better expressed through equity, bonds, loans, or no security yet. Do not recommend credit without current price, yield, spread, maturity, and covenant context.

Output:

| Issue | Equity implication | Credit implication | Current evidence quality |
| --- | --- | --- | --- |
|  |  |  |  |

### Valuation Analyst Checks Whether It Is Priced In

Prompt:

Review all agent outputs. Identify which parts of the thesis may already be priced in. Explain what comparison or market data is required before calling the security attractive.

Output:

| Thesis point | Could be priced in? | Data needed | Decision impact |
| --- | --- | --- | --- |
|  | Yes / No / Unknown |  |  |

### Market Analyst Checks Base Rate

Prompt:

Review the debate. Identify which risks or opportunities are company-specific versus sector-wide. Explain whether the company is actually unusual or simply reflecting the sector setup.

Output:

| Topic | Company-specific or sector-wide | Evidence | Diligence needed |
| --- | --- | --- | --- |
|  |  |  |  |
